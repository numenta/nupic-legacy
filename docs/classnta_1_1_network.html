Detailed Description

A Network represents an HTM network. A network is a collection of regions.

Member Typedef Documentation

typedef void(* nta::Network::runCallbackFunction)(Network *, UInt64 iteration, void *)

You can attach a callback function to a network, and the callback function is called after every iteration of run(). To add a callback, just get a reference to the callback collection with getCallbacks, and add a callback
